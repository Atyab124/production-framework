## Bootstrap deviation note

This release was implemented via CTO main-session edits, not Builder dispatch. The Builder agent's `isolation: worktree` interacts badly with the framework's own dev environment (F-V21). Once the Builder fixes (F-V10 empty-diff gate, F-V20 inheritance) prove reliable in a real project (Taskforge or other), the next release should run dogfood-via-Builder.

The dogfood gate in `docs/release-discipline.md` is acknowledged as not-fully-runnable for this specific release. Future releases run it.
